# Archon Phase 2 — Rethlas Informal Proof

Dual-agent system (Generation + Verification) for informal mathematical proof.

**Capabilities:**
- Web search + theorem retrieval (Matlas, Mathlib, Loogle, LeanSearch)
- Construct examples / counterexamples  
- Propose decomposition plans
- Parallel recursive proof exploration (subagent)
- Identify failure patterns
- Verify proof correctness

**Output:** Candidate informal proof + verification report.
